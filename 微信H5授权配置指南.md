## 公众号后台配置步骤

### 第1步：配置网页授权域名
1. 登录 [微信公众平台](https://mp.weixin.qq.com)
2. 进入 **设置与开发 → 公众号设置 → 功能设置**
3. 找到 **网页授权域名**，点击"设置"
4. 下载校验文件（类似 `MP_verify_xxxxx.txt`）
5. 把校验文件发给我，我上传到服务器 `/var/www/linmeng/client/`
6. 验证通过后，填写：`www.3qall.com`

### 第2步：确认 AppID 和 AppSecret
- AppID：`wxa382e1c9fb93780e`（代码中已配置）
- AppSecret：需要在公众号后台 **设置与开发 → 基本配置 → 开发者Secret** 获取
- 获取到后，在服务器 `server/.env` 中添加：
  ```
  WECHAT_PUBLIC_SECRET=你的公众号AppSecret
  ```

## 使用方式

### 在微信内打开 H5
用户点击公众号菜单 或 在微信里打开链接：
```
https://www.3qall.com/#/wechat-login
```

### 授权流程
1. 自动跳转到微信授权页
2. 用户点击"确认授权"
3. 回调到 `/#/wechat-login?code=xxx`
4. 后端用 code 换 openid/unionid
5. 如果已绑定账号 → 直接登录
6. 如果未绑定 → 引导用户绑定手机号

---

## 后端接口说明

| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/wechat/h5-auth` | POST | H5 微信授权，用 code 换登录信息 |
| `/api/wechat/h5-bind-phone` | POST | 绑定手机号（新用户） |

### 请求示例
```json
// h5-auth
POST /api/wechat/h5-auth
{ "code": "微信返回的code" }

// 返回（已绑定）
{
  "code": 0,
  "data": {
    "isNew": false,
    "token": "...",
    "userType": "community",
    "userId": 1,
    "userName": "张三"
  }
}

// 返回（未绑定）
{
  "code": 0,
  "data": {
    "isNew": true,
    "openid": "oXXX",
    "unionid": "oXXX",
    "nickname": "微信用户",
    "avatar": "http://..."
  }
}
```
