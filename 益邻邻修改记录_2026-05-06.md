### 4. 新建数据表

#### publisher_followers 表
```sql
CREATE TABLE publisher_followers (
  id INT AUTO_INCREMENT PRIMARY KEY,
  publisher_id INT NOT NULL,
  user_id INT NOT NULL,
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  UNIQUE KEY uk_publisher_user (publisher_id, user_id)
);
```

#### activity_sessions 表
```sql
CREATE TABLE activity_sessions (
  id INT AUTO_INCREMENT PRIMARY KEY,
  activity_id INT NOT NULL,
  venue_id INT,
  session_title VARCHAR(100),
  session_name VARCHAR(100),
  type VARCHAR(50),
  start_time DATETIME,
  end_time DATETIME,
  max_participants INT DEFAULT 0,
  current_participants INT DEFAULT 0,
  status VARCHAR(20) DEFAULT 'scheduled',
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  INDEX idx_activity (activity_id),
  INDEX idx_venue (venue_id)
);
```

#### venue_bookings 表
```sql
CREATE TABLE venue_bookings (
  id INT AUTO_INCREMENT PRIMARY KEY,
  venue_id INT NOT NULL,
  user_id INT NOT NULL,
  session_id INT,
  booking_date DATE,
  start_time TIME,
  end_time TIME,
  status VARCHAR(20) DEFAULT 'pending',
  notes TEXT,
  created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  INDEX idx_venue (venue_id),
  INDEX idx_user (user_id)
);
```

## 三、后端代码修复

### 1. authController.js - gender 字段修复

**文件：** `/opt/yilinlin/backend/src/controllers/authController.js`

**问题：** `user.gender || 0` 会返回数字0，与数据库 ENUM 类型不匹配

**修复：**
```javascript
// 修复前
gender: user.gender || 0

// 修复后
gender: user.gender || 'unknown'
```

**涉及位置：**
- 普通登录响应（第308行）
- 微信登录响应（第230行）
