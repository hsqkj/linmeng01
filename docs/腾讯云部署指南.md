## 第一步：注册腾讯云账号

### 1.1 访问腾讯云官网

### 2.1 进入云服务器购买页

https://buy.cloud.tencent.com/cvm

| **内存** | 4GB |

| **系统盘** | 50GB SSD 云硬盘 |
| **公网带宽** | 按带宽计费，5Mbps |
| **购买时长** | 1年（享折扣） |

```

```

### 2.4 设置登录密码
- 设置 root 密码（**务必记住！**）
- 或绑定SSH密钥（更安全）

### 2.5 完成购买
- 确认订单 → 支付

## 第三步：获取服务器信息

### 3.1 登录控制台查看
进入「云服务器」→ 点击你的实例，查看：
- **公网IP**（如：123.456.789.10）
- **端口号**：默认22（SSH）

### 3.2 配置安全组（重要！）
1. 点击实例 → 防火墙 → 添加规则
2. 放行以下端口：

| 端口 | 用途 | 协议 |
|------|------|------|
| 22 | SSH远程登录 | TCP |
| 80 | HTTP网站访问 | TCP |
| 443 | HTTPS安全访问 | TCP |
| 3000 | 后端API（可选限制） | TCP |
| 3306 | MySQL（仅内网） | TCP |

```bash

curl -fsSL https://deb.nodesource.com/setup_18.x | bash -
apt install -y nodejs

# 验证安装
node -v  # 应显示 v18.x.x

```

### 5.2 安装 MySQL 8.0
```bash
# Ubuntu

# 启动MySQL
systemctl start mysql
systemctl enable mysql

# 安全配置（设置root密码）
mysql_secure_installation
```

### 5.3 安装 Nginx
```bash
apt install -y nginx

# 启动Nginx
systemctl start nginx
systemctl enable nginx
```

### 5.4 安装 PM2（进程管理器）
```bash
npm install -g pm2

# 验证安装
pm2 -v
```

## 第六步：配置数据库

### 6.1 登录MySQL
```bash
mysql -u root -p
# 输入之前设置的密码
```

### 6.2 创建数据库和用户
```sql
-- 创建数据库
CREATE DATABASE linmeng CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;

-- 创建用户（修改密码为强密码）
CREATE USER 'linmeng'@'localhost' IDENTIFIED BY '你的数据库密码';

-- 授权
GRANT ALL PRIVILEGES ON linmeng.* TO 'linmeng'@'localhost';
FLUSH PRIVILEGES;

-- 退出
EXIT;
```

**⚠️ 记住这些信息：**
- 数据库名：`linmeng`
- 数据库用户：`linmeng`
- 数据库密码：`xxxxxx`

## 第十步：配置Nginx

### 10.1 创建Nginx配置文件
```bash
vim /etc/nginx/sites-available/linmeng
```

添加以下配置：
```nginx
server {
    listen 80;
    server_name 3qall.com www.3qall.com;  # 你的域名

    # 前端静态文件
    root /var/www/linmeng/client/dist;
    index index.html;

    # 前端路由
    location / {
        try_files $uri $uri/ /index.html;
    }

    # 后端API代理
    location /api {
        proxy_pass http://localhost:3000;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection 'upgrade';
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_cache_bypass $http_upgrade;
    }

    # 静态资源缓存
    location ~* \.(js|css|png|jpg|jpeg|gif|ico|svg|woff|woff2)$ {
        expires 30d;
        add_header Cache-Control "public, immutable";
    }
}
```

### 10.2 启用配置
```bash
# 创建软链接
ln -s /etc/nginx/sites-available/linmeng /etc/nginx/sites-enabled/

# 测试配置
nginx -t

# 重载Nginx
systemctl reload nginx
```

## 常见问题

### Q1: 服务器连不上？
- 检查安全组是否放行22端口
- 检查本地网络是否限制SSH

### Q2: 数据库连接失败？
- 检查MySQL是否运行：`systemctl status mysql`
- 检查用户名密码是否正确

### Q3: 域名无法访问？
- 检查DNS解析是否生效：`ping 3qall.com`
- 检查Nginx配置是否正确

### Q4: SSL证书申请失败？
- 确保域名已解析到服务器IP
- 确保80端口开放
