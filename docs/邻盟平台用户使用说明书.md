## 1. 平台概述

### 1.1 什么是邻盟

**邻盟**（LinMeng）是一个社区互助资源对接平台，致力于连接社区与商家资源，促进社区服务的共建共享。

平台围绕"需求发布 → 资源匹配 → 合作撮合 → 评价激励"的全链路，为社区、商家、招商大使三方提供一站式服务。

### 1.2 三大用户端

| 端口 | 适用对象 | 核心价值 |
|------|---------|---------|
| **社区端** | 社区管理者、居委会 | 发布需求、寻找资源、管理合作 |
| **商家端** | 企业商家、社会组织 | 发布资源、匹配需求、拓展渠道 |
| **招商大使端** | 平台推广人员 | 发展商家入驻、获取提成奖励 |

## 2. 账号登录与切换

### 2.1 登录方式

在平台首页右上角点击 **登录**，输入手机号和密码即可登录。

### 2.2 账号类型切换

登录后，顶部导航栏会显示当前所在端：
- **社区端**：默认入口，`/#/community`
- **商家端**：`/#/merchant`
- **招商大使端**：`/#/ambassador`

> 💡 **提示**：同一手机号可同时注册多个角色（如同时是社区账号又是商家账号），在顶部切换端口即可。

### 2.3 找回密码

在登录页点击"忘记密码"，通过绑定的手机号接收验证码完成密码重置。

## 3. 社区端使用指南

**入口**：`http://www.3qall.com/#/community`

### 3.1 首页

社区端首页展示以下核心内容：

| 模块 | 说明 |
|------|------|
| **欢迎横幅** | 显示社区名称、社区编码、区域位置 |
| **今日数据** | 需求数量、资源数量、撮合成功数 |
| **为你推荐** | AI算法推荐的相关资源卡片 |
| **发布需求** | 快捷发布社区需求 |

#### 发布需求（4步向导）

点击首页「发布需求」按钮或导航栏「发布需求」，按以下步骤操作：

**第1步：选择需求类型**

| 类型 | 说明 | 附加字段 |
|------|------|---------|
| 🎭 活动合作 | 社区文化、体育、教育活动 | 活动类型、人数规模、活动时间、预算等 |
| 👨‍🏫 专家需求 | 专业技术指导 | 专家类型、服务次数、是否需要上门 |
| 🏠 场地需求 | 活动空间需求 | 场地名称、面积要求、设施需求、开放时间等 |

**第2步：填写详细信息**

- 需求标题（必填）
- 详细描述（必填，含富文本编辑器）
- 期望回报类型（多选）：资金支持 / 物资支持 / 人力服务 / 专业服务 / 媒体报道
- 期望回报说明

**第3步：设置回报内容**

- 合作方式：现金支付 / 资源置换 / 联合主办 / 公益合作
- 品牌展示类型：横幅露出 / 物料赞助 / 媒体宣传 / 其他
- 积分激励：自愿提供社区积分作为答谢

**第4步：确认并提交**

- 检查以上信息
- 点击「提交审核」，等待管理员审核后上线

> 📝 **草稿保存**：发布过程中可点击「保存草稿」，下次进入自动恢复。
> ⏳ **审核说明**：提交后需管理员审核，审核状态可在「我的需求 → 草稿箱」中查看。

### 3.2 资源广场

路径：导航栏「资源广场」

#### 功能说明

- **类型筛选**：专业服务 / 教育培训 / 场地资源 / 物资捐赠 / 志愿服务 / 资金赞助 / 技术支持 / 健康医疗 / 活动赞助 / 媒体宣传 / 技能培训 / 养老服务
- **搜索**：按关键词搜索资源标题和内容
- **收藏**：点击资源卡片上的收藏图标，已登录用户可将资源收藏至个人中心
- **分页加载**：每次加载12条资源

#### 查看资源详情

点击任意资源卡片进入详情页，详情页展示：

- **基础信息**：标题、资源类型、标签、发布时间
- **可提供内容**：按资源类型动态显示（金额范围 / 物资数量 / 服务人数 / 资质证明等）
- **商家信息**：商家名称、地址、Logo、图文介绍
- **地图位置**：商家地理位置展示
- **评论区**：其他社区的评价与留言
- **操作按钮**：收藏 / 关注 / 分享 / 发起合作意向

> 💡 **快速发起合作**：在资源详情页点击「发起合作意向」按钮，可直接向商家发送合作请求。

### 3.3 需求广场

路径：导航栏「需求广场」

展示社区发布的需求，可按类型筛选、关键词搜索，快速找到与本社区匹配的需求信息，支持收藏。

### 3.4 个人中心

路径：右上角头像 →「个人中心」

#### 7大功能模块

| 模块 | 说明 |
|------|------|
| **基本信息** | 社区名称、编码、区街社区地址（不可修改）、小区名称、联系人、手机号、邮箱 |
| **修改密码** | 验证当前密码后设置新密码 |
| **社区画像** | 社区类型（老旧小区/新建社区/亲子社区等）、居民类型（青少年/儿童/退役军人等）|
| **我的标签** | 管理社区的兴趣标签和服务需求标签 |
| **我的收藏** | 收藏的资源列表，支持取消收藏 |
| **我的奖励** | 平台积分奖励记录 |
| **退出登录** | 安全退出当前账号 |

> 🔒 **地址保护**：区、街、社区名称在注册后不可修改，如需调整请联系管理员。

### 3.5 消息中心

路径：导航栏「消息」图标

#### 三类消息

| 类型 | 说明 |
|------|------|
| 🔔 **系统通知** | 平台公告、审核结果、活动通知 |
| 🤝 **合作意向** | 商家发来的合作请求与回复（可查看/回复/忽略） |
| 💬 **留言咨询** | 其他社区或用户的直接留言 |

> 💡 **合作意向处理**：当商家对某需求有意向时，会在此收到通知，可选择接受或婉拒。

点击「发布资源」进入发布向导：

**第1步：选择资源类型**

> 📌 **说明**：资源类型由管理后台统一配置，共有9种基础类型。以下为平台默认类型：

| 类型 | 说明 |
|------|------|
| 专业服务 | 咨询、培训、设计、法律、财税等 |

| 场地资源 | 活动空间和场地支持 |
| 物资捐赠 | 实物物资的捐赠或支持 |
| 志愿服务 | 社区志愿活动 |
| 资金赞助 | 提供资金支持 |
| 技术支持 | IT、技术解决方案等 |
| 健康医疗 | 医疗健康相关服务 |
| 活动赞助 | 活动举办的支持 |

> 💡 **完整类型列表**：所有资源类型以管理后台「基础数据」中的配置为准。

**第2步：填写详细信息**

根据选择的类型，显示对应字段：

| 资源类型 | 典型字段 |
|---------|---------|
| 资金赞助 | 金额范围、赞助场景 |
| 物资支持 | 物资数量、领取方式 |
| 人力服务 | 派遣人数、服务时长、服务范围 |
| 专业服务 | 服务类型（咨询/培训/设计/法律/财税/IT/健康/其他）、服务范围、资质证明、收费标准 |
| 媒体宣传 | 媒体类型（图文/视频/直播/其他）、覆盖范围 |
| 其他 | 自定义描述 |

> 📝 **共同字段**：标题、详细描述、标签、省市区地址、详细地址、经纬度坐标、图片上传

**第3步：设置期望回报**

- 选择期望回报类型（可多选）
- 填写期望回报说明，向社区说明希望获得何种回报

**第4步：确认并提交**

- AI智能审核（自动敏感词/广告词检测）
- 审核通过后资源上线，进入资源广场展示

> 📝 **草稿保存**：发布过程中可保存草稿，下次进入自动恢复。草稿在「资源管理 → 草稿箱」中管理。

**入口**：`http://www.3qall.com/#/ambassador`

### 5.1 首页

| 模块 | 说明 |
|------|------|
| **欢迎横幅** | 头像 + 大使姓名 + 渠道码 + 入职时间 |

| **收益概况** | 累计提成 / 待结算 / 已提现 / 账户余额 |
| **快速操作** | 分享渠道码 / 发展记录 / 提成明细 / 申请提现 |

### 5.2 我的渠道码

路径：首页「查看我的二维码」或导航栏「渠道码」

#### 页面内容

- **专属二维码卡片**：含品牌Logo、大使姓名、二维码图形、渠道码
- **分享按钮**：下载二维码图片 / 复制注册链接
- **专属注册链接**：商家通过此链接注册，系统自动绑定大使关系
- **使用提示**：
  - 分享二维码给商家 → 商家扫码注册 → 缴纳会费后自动计算提成
  - 绑定关系**永久有效**，商家每次续费都可获得续费提成
  - 提成**每月1日**自动结算，满100元可申请提现
- **小统计数据**：累计发展 / 累计提成 / 待结算

> 📤 **推广技巧**：将二维码印制在名片、宣传单、易拉宝等物料上，在社区活动或商会会议上推广。

### 5.3 发展记录

路径：导航栏「发展记录」

展示所有通过大使渠道码注册的商家列表：

| 字段 | 说明 |
|------|------|
| 商家名称 | 注册的公司/组织名称 |
| 会员等级 | 注册时选择的会员等级（免费试用/普通/银牌/金牌/铂金/钻石）|
| 入驻时间 | 注册日期 |
| 状态 | 待审核 / 已缴费 / 禁用 |
| 操作 | 查看详情 |

### 5.4 提成明细

路径：导航栏「提成明细」

- 按月筛选提成记录
- 显示提成类型：**首次入会提成** / **续费提成**
- 显示对应商家和金额

#### 提成政策（根据管理后台设置）

| 项目 | 说明 |
|------|------|
| 首次入会提成 | 根据管理后台的「提成设置」配置 |
| 续费提成 | 根据管理后台的「提成设置」配置 |
| 最低提现金额 | 100元 |
| 结算周期 | 每月1日自动结算 |

> 💰 **提成计算示例**：提成比例由管理员在管理后台配置，实际提成金额 = 商家缴纳会费 × 提成比例。具体比例请咨询管理员或查看管理后台的提成设置页面。

### 5.5 提现管理

路径：导航栏「提现」

#### 设置收款账户

首次提现前需设置收款账户，支持三种方式：

| 方式 | 必填信息 |
|------|---------|
| 🏦 银行卡 | 开户行、银行账号、开户姓名 |
| 💚 支付宝 | 支付宝账号、真实姓名 |
| 💙 微信 | 微信账号（已绑定本手机号自动获取）|

#### 申请提现

- 选择收款账户
- 输入提现金额（不低于最低提现额度）
- 确认后提交，管理员审核后打款

#### 提现记录

查看历史提现记录：申请时间 / 金额 / 收款账户 / 状态（待审核/已打款/已拒绝）

## 7. 常见问题与使用技巧

### 7.1 账号相关

**Q：忘记密码怎么办？**
> A：在登录页点击「忘记密码」，输入手机号获取验证码后重置密码。

**Q：收不到验证码怎么办？**
> A：请检查手机信号和网络状态，确保手机未欠费或拉黑平台号码。验证码有效期为5分钟，若仍未收到，可点击「重新获取」再次尝试。平台发送的是真实短信验证码，请注意查收。

**Q：注册后无法登录提示"账号不存在"？**
> A：检查手机号是否正确，或联系平台管理员确认账号状态。

**Q：同一手机号可以注册多个角色吗？**
> A：可以。同一手机号可同时拥有社区账号、商家账号和招商大使账号，在平台顶部切换端口即可。

### 7.2 发布相关

**Q：发布需求/资源需要审核吗？**
> A：是的，所有发布的内容需管理员审核后上线。审核状态可在草稿箱中查看。

**Q：草稿会自动保存吗？**
> A：发布过程中点击「保存草稿」，下次进入时自动恢复。

**Q：发布后可以修改内容吗？**
> A：可以。在「资源管理」或「我的需求」中找到已发布内容，点击编辑修改。

### 7.3 合作相关

**Q：如何与商家/社区建立合作？**
> A：进入资源详情页 → 点击「发起合作意向」→ 填写合作说明 → 等待对方回复。

**Q：收到合作意向后如何处理？**
> A：进入消息中心 → 打开「合作意向」消息 → 选择接受 / 婉拒 / 查看详情。

**Q：撮合成功后有奖励吗？**
> A：平台设有撮合奖励机制。每成功撮合一笔合作，平台向社区捐赠价值50元的物资，商家可获得相应的撮合积分奖励。具体奖励规则可在个人中心「我的奖励」中查看，或咨询管理员了解管理后台的撮合奖励配置。

### 7.4 招商大使相关

**Q：提成多久结算一次？**
> A：每月1日自动结算上月的待结算提成。

**Q：提成最低多少可以提现？**
> A：默认最低提现金额为100元。

**Q：商家续费我能获得提成吗？**
> A：可以。绑定关系永久有效，商家每次续费都会按续费提成比例计算您的提成。

**Q：推广链接和二维码在哪里获取？**
> A：审核通过后，进入「我的渠道码」页面即可下载二维码和复制专属注册链接。

### 7.5 移动端使用

**Q：平台支持手机访问吗？**
> A：支持。平台已做响应式设计， 手机浏览器访问体验流畅。

**Q：手机端发布内容方便吗？**
> A：手机端支持发布资源/需求、查看消息、管理收藏等全部核心功能。

### 7.6 数据安全

**Q：我的社区/商家信息会被泄露吗？**
> A：不会。平台严格保护用户隐私数据，仅在合作双方同意后交换联系方式。

**Q：如何修改绑定的手机号？**
> A：请联系平台管理员，提供身份证明后协助修改。
