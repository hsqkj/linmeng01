## 问题诊断与修复

### 问题描述
商家发布资源时，商家类型选择"专业服务"后，填写的字段（专业服务类型、服务范围、资质证明、收费标准等）在社区端详情页没有正确显示，反而显示的是资金赞助/物资捐赠/人力支持等不相关的字段。

### 问题根因
前端 PublishResource.vue 填写的很多字段**没有提交到后端**，导致数据库中这些字段为空。

### 修复内容

#### 1. 数据库字段扩展
**resources 表新增字段：**
- `professional_type` - 专业服务类型
- `tech_types` - 技术类型（JSON数组）
- `tech_service_type` - 技术服务方式

## 数据库操作
使用以下SQL命令添加字段：
```sql
-- resources 表
ALTER TABLE resources ADD COLUMN professional_type VARCHAR(100) AFTER coverage;
ALTER TABLE resources ADD COLUMN tech_types JSON AFTER certification;
ALTER TABLE resources ADD COLUMN tech_service_type VARCHAR(50) AFTER tech_types;
ALTER TABLE resources ADD COLUMN goods_expiry DATE AFTER tech_service_type;
ALTER TABLE resources ADD COLUMN fund_scenes JSON AFTER price_range;
ALTER TABLE resources ADD COLUMN media_channels JSON AFTER media_type;
ALTER TABLE resources ADD COLUMN valid_until DATE AFTER expected_reward_desc;

-- demands 表
ALTER TABLE demands ADD COLUMN expert_count INT AFTER activity_type;
-- ... 其他字段类似
```
